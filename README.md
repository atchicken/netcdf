# netcdf
Create NetCDF File for Python

## How to Use
```bash:bash
python createNetCDF.py -n {NetCDF Kind} -p {Save NetCDF File Path}
```

### NetCDF Kind
- 1: NetCDF4
- 2: NetCDF4 Classic
- 3: NetCDF3 Classic
- 4: NetCDF3 64Bit_Offset
- 5: NetCDF3 64Bit_Data
- Others: NetCDF
